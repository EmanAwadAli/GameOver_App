# ☠️ Game Over App
## 📌 Overview
In this project, I create a web application that display a set of games and its related details.

## 📥 Installation
###### Clone Project
```
git clone https://github.com/EmanAwadAli/GameOver_App.git
```
###### Install npm packages
```
npm i
```
###### Start your Project
```
npm run dev
```
