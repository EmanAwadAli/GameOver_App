# ☠️ GameOver App
## 📌 Overview
In this project, I created a web application that displays a set of games and its related details.

## 📥 Installation

###### Clone Project
```
git clone https://github.com/EmanAwadAli/GameOver_App.git
```
###### Install npm packages
```
npm i
```
###### Start your Project
```
npm run dev
```
